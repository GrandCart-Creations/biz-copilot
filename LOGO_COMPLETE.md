## ğŸ’¡ Logo Customization

Want to tweak the logo? Here's how:

### Change Colors
1. Open SVG file in text editor
2. Find gradient definition
3. Replace hex codes:
   - #4F46E5 (Indigo)
   - #9333EA (Purple)
   - #EC4899 (Pink)

### Adjust Roundness
1. Find `<rect>` tag
2. Change `rx="45"` value
   - Higher = more rounded
   - Lower = sharper corners

### Modify Text Size
1. Find `<text>` tag
2. Change `font-size="85"`
3. Adjust as needed
